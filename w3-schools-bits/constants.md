# A detailed summary of Go constants
A Go constants handles function where a variable have a fixed value that can not be changed.
where const keyword is declared it shows that the variable is readeable.when a constant is declared it looks like this:
const LA = 12.2
func main() {
    fmt.Println(LA)
}

Constants have 2 types which are:
1. typed constant is declared with a type
2. untyped constant is declared without type.
