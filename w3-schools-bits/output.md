# A detailed summary of output
A Go output function handles all the three functions that prints arguments in ddefault, newline at the end, and formatting verb such as %value, %type. the consist of print() println() printf()

the general formatting verbs handles data types with descriptions such as %v, %#v, %T, %%.
There are other types of formatting verbs such as:
integer formatting verb%b\n,string formatting verb%s, boolean formatting verb%t\n, float formatting verb%f.
