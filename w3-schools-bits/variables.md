# A detailed summary of variable
A Go variable handles data value storage that operate whith different types such as:
1. string that stores values of text with double quotes.
2. bool stores values such as true/false.
3. integer store values in numbers such as positive or negative numbers
4. float64 stores values in decimalpoints such as 10.22
